# Concepts learnt and applied in this project
- **Variables & Types:** `int` (whole numbers), `double` (decimals), `char` (single characters), `bool` (true/false states), `std::string` (text).
- **Output:** `std::cout` and chaining output with `<<`.
- **Constants:** The `const` keyword makes a variable read-only. Standard convention is `ALL_CAPS` for constant names.
- **Namespaces:** `namespace Name { ... }` blocks used to prevent naming conflicts (e.g., having a `gravity` variable for two different planets). Access them with `::` (e.g., `Earth::gravity`).
- **Type Aliases:** `typedef` and `using`. These create nicknames for data types (e.g., calling a `double` a `kilograms_t` for clarity).
